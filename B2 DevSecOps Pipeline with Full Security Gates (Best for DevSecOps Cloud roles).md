Build a real CI/CD pipeline that a company would actually use:

- GitHub Actions → Trivy (container scan) → SonarQube (SAST) → OWASP ZAP (DAST) → Deploy to EC2
- Terraform for infra, Docker for containerization
- Block deployments automatically if critical CVEs found

**Why it wins:** You already have pieces of this in MindCraft — but a standalone, security-gate-focused version with DAST added makes it a completely different story. Very aligned with DevSecOps job descriptions.
